# AI-PLUG EXPORTER 2.0 (Beta)

## Description

**AI-PLUG EXPORTER 2.0** is a plugin for Adobe Illustrator that enables easy and precise export of selected artboards as individual PDF or AI files. It simplifies working with documents containing multiple artboards, providing greater control and efficiency.

## Installation

1. **Download** the plugin from [GitHub](insert_link_here) or [external link](insert_link_here) to any folder on your PC.

## Usage

1. **Open** a project with multiple artboards (AI or PDF) in Adobe Illustrator.
2. **Run the plugin:**
   - Press the shortcut **Ctrl + F12**.
   - Point to the file AI-PLUG EXPORTER 2.0.js on your computer.
3. **Select** the artboards you want to export.
4. **Configure** export options:
   - Choose the export format (PDF or AI).
   - Check the option **Open destination folder** before or after export, if desired.
5. Click **Export** to start the process.

## Features

- **Precise export** of selected artboards as individual files.
- **User-friendly interface** for easy configuration.
- **Shortcut key** for quick access.
- **Additional options**, such as opening the destination folder.

## Requirements

- Adobe Illustrator CS6 or later.
- Windows or macOS operating system.
